This program demonstrates client and server communication to find type and package dependencies

How to start the project

1) Compile the application using compile.bat file present in the root folder
2) Start the server application using the file runServerExecutive.bat
3) Start the client application using the file runClientExecutive.bat
4) Enter the relevant details as required
5) If multiple server or multiple clients needs to be launched then please execute the corresponding bat files

*Note: The files have to be launched in administrator mode to enable the client-server communication
